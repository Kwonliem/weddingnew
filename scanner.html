QR Code Check-in Tamu

``` ### Cara Kerjanya 1. **Panitia Membuka `scanner.html`:** Panitia membuka file ini di browser HP mereka (misalnya, Google Chrome di Android). Browser akan meminta izin untuk menggunakan kamera. 2. **Scan QR Code Tamu:** Panitia mengarahkan kamera ke QR Code yang ada di e-ticket PDF tamu. 3. **Pengiriman Data:** Setelah QR Code terbaca, halaman scanner akan otomatis mengirimkan ID tamu yang ada di dalam QR Code ke URL Google Apps Script Anda. 4. **Backend Bekerja:** * Fungsi `doGet` di dalam Apps Script Anda akan aktif. * Script akan mencari `GuestID` tersebut di Google Sheet. * Jika ditemukan, script akan mengubah tulisan di kolom `StatusCheckIn` dari "Belum di Scan" menjadi **"Sudah di Scan"**. * Script akan mengirim balik pesan sukses (misalnya, "Sukses! Musalim Yusuf berhasil di-check-in."). 5. **Hasil Tampil di HP Panitia:** Pesan sukses tersebut akan ditampilkan di layar HP panitia, dan scanner siap untuk memindai tamu berikutn
